About Task Management System

Welcome to the Task Management System, a web application developed using Django, Python, Bootstrap, and SQLite.

This application helps users organize and manage their daily tasks in a simple and efficient way. Users can create new tasks, update existing tasks, mark tasks as completed, and delete unnecessary tasks.

✨ Key Features

  • Create new tasks
  • View all tasks
  • Edit existing tasks
  • Mark tasks as completed
  • Delete completed tasks
  • Simple and user-friendly interface

🛠 Technologies Used

Frontend HTML5, CSS3, Bootstrap
Backend Python, Django
Database SQLite
Language Python

CRUD Operations

Operation Description
Create Add a new task.
Read View all available tasks.
Update Edit an existing task.
Delete Remove a task permanently.
Complete Mark a task as completed.

Why Use This Application?

  • Improve productivity.
  • Manage daily tasks efficiently.
  • Easy to use interface.
  • Fast and lightweight application.
  • Learn Django CRUD operations.
