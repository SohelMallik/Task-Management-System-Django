About This Project

This Task Management System is built using Django, Python, Bootstrap, and MySQL.

The application allows users to:

  • Create new tasks.
  • View all tasks.
  • Edit existing tasks.
  • Mark tasks as completed.
  • Delete tasks.

This project demonstrates CRUD (Create, Read, Update, Delete) operations using the Django framework.
